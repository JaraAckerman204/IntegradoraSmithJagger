git init - Inicilizar el repositorio 
git status - Verificar que cambios se realizaron en la carpeta "Local"
git add . - Añades todos los cambios o archivos /  git add (ejemplo.html) - Añades un cambio o archivos en especifico
git commit -m "Hice un nuevo cambio" - Poner un comentario a un cambio
git push -u origin main - Subir los cambios a la rama main para repositorio "Remoto"
git pull - Descargar los cambios que hicieron otros miembros.
